# 💼 Proyecto Módulo 8: Desarrollo de un Portafolio Digital

## 📌 Descripción
Este proyecto tiene como objetivo preparar un perfil profesional para el mercado laboral IT, mediante la creación de un portafolio técnico y el desarrollo de habilidades de empleabilidad.

Se enfoca en mostrar conocimientos, proyectos desarrollados y la capacidad de aportar valor dentro de una organización tecnológica.

---

## 🎯 Objetivo del Proyecto
Desarrollar un portafolio digital que permita:

- Mostrar habilidades técnicas adquiridas
- Presentar proyectos realizados durante el curso
- Destacar el perfil profesional
- Prepararse para procesos de selección en el área tecnológica

---

## 👨‍💻 Contexto
El proyecto simula un proceso de selección para el cargo de:

**Talento Junior Tech (Trainee/Junior)**

Donde se espera:
- Participación en proyectos tecnológicos
- Trabajo en equipo
- Propuesta de soluciones técnicas
- Documentación de procesos
- Aprendizaje continuo

---

## 🧠 Habilidades Aplicadas

- Desarrollo web
- Programación (Python, SQL, etc.)
- Uso de metodologías ágiles
- Trabajo colaborativo
- Resolución de problemas
- Comunicación efectiva

---

## 📂 Contenido del Proyecto

Este repositorio incluye:

- 📄 Portafolio digital
- 📄 Caso de estudio técnico
- 📄 Investigación de empresa
- 📄 Documento PDF entregable
- 📄 Código fuente de proyectos desarrollados

---

## 🔍 Actividades Realizadas

### 1. Investigación de Empresa
- Análisis de una empresa del sector tecnológico
- Evaluación de:
  - Valores y propósito
  - Tecnologías utilizadas
  - Metodologías de trabajo
- Identificación de aportes personales al equipo

---

### 2. Desarrollo del Portafolio
- Creación/actualización del portafolio digital
- Inclusión de al menos 3 proyectos
- Presentación de un caso de estudio que incluye:
  - Descripción del problema
  - Solución implementada
  - Herramientas utilizadas
  - Resultados obtenidos
  - Aprendizajes

---

### 3. Análisis FODA (PLUS)
- Fortalezas
- Oportunidades
- Debilidades
- Amenazas

---

## 📊 Entregables

- Documento en formato PDF
- Portafolio digital online
- Caso de estudio documentado

---

## 🚀 Tecnologías Utilizadas

- Bases de datos (SQLite / SQL)
- Herramientas de desarrollo web

Este proyecto busca fortalecer la empleabilidad en el área tecnológica, permitiendo:

- Mejor presentación profesional
